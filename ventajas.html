¿Por qué utilizar bootstrap?

Actualmente bootstrap es el segundo proyecto mas destacado de github, e incluso es utilizado por la NASA. Entonces, ¿por qué se usa tanto?

  • Permite crear sitios adaptables
  • Fácil de usar y personalizar
  • Mantenimiento y actualización
  • Compatible con los navegadores más utilizados

Desventaja

Como todo en la vida, no todo son ventajas en el uso de frameworks de diseño. Existen muchos detractores de Bootstrap y los motivos son varios. A nuestro juicio, ninguno de ellos es realmente definitivo, pero creemos que es importante señalar los principales puntos en contra.

  • Optimización. Bootstrap contiene mucho código para componentes, iconos, rejillas, etc.
  • Código HTML complejo. Nuestro código HTML tendrá que ser modificado para crear los contenedores tal como Bootstrap necesita para cargar en ellos ciertos componentes.
  • Diseños muy similares entre sí. Si no personalizamos el diseño de nuestro sitio obtendremos un diseño bastante aparente, pero infelizmente parecido a otros millares de sitios de Internet.

Por eso es importante tener en cuenta estas desventajas para saber si utilizar o no bootstrap.
